Para aumentar la cantidad de transacciones, primero debe ocurrir un escenario complementario para la oferta y la demanda; esta relacion tiene que empezar por la manifestación del interes, a traves del envio de ofertas.

Una transaccion tiene lugar cuando un usuario interesado -demanda- envia una oferta de trueque, es decir, envia una propuesta de un item al usuario con el item de interes -oferta- . Este ultimo deberá aceptarla para iniciar el trueque, es posible que surja una negociación previo a la aceptación de la oferta.

Ahora bien, guiados por la pregunta analitica nuestro interes es predecir si un usuario aceptará una oferta de trueque. Para esto clasificaremos por usuarios, a partir de una variables predictoras, si para un dia especifico el usuarios aceptara o no una oferta de trueque.

La regresión logística es un método de análisis estadístico que se utiliza comúnmente en la modelización de datos binarios o de variables dependientes categóricas. Se utiliza para predecir la probabilidad de que una observación pertenezca a una de dos categorías, y es especialmente útil cuando se trata de datos que no siguen una distribución normal o cuando la relación entre las variables independientes y la variable dependiente no es lineal.

Matriz de confusión

Una matriz de confusión es una tabla que se utiliza para evaluar el desempeño de un modelo de clasificación en un conjunto de datos.
La matriz de confusión tiene una estructura cuadrada con el mismo número de filas y columnas. En el caso de una clasificación binaria, como sí/no o positivo/negativo, la matriz de confusión tiene cuatro celdas: Verdaderos Positivos (VP), Falsos Positivos (FP), Falsos Negativos (FN) y Verdaderos Negativos (VN).

• Verdaderos Positivos (VP): el modelo predice correctamente la clase positiva.

• Falsos Positivos (FP): el modelo predice incorrectamente la clase positiva.

• Falsos Negativos (FN): el modelo predice incorrectamente la clase negativa.

• Verdaderos Negativos (VN): el modelo predice correctamente la clase negativa.

La importancia de la matriz de confusión radica en que permite evaluar la precisión y el rendimiento de un modelo de clasificación. A partir de los valores en la matriz de confusión, se pueden calcular varias métricas, como la precisión, la sensibilidad, la especificidad y la tasa de error, que ayudan a entender cómo de bien está funcionando el modelo.

La matriz de confusión es una herramienta importante para evaluar el rendimiento de un modelo de clasificación y para identificar las áreas en las que se puede mejorar el modelo.
